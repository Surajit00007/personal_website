## 🛠️ Local Setup

```bash
npm install
```

Create a `.env` file in the root:
```env
GROQ_API_KEY=your_groq_api_key_here
```

Run locally:
```bash
npm run dev
```

Open **[http://localhost:8080](http://localhost:8080)**
